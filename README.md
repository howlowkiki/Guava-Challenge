# 🍐 Guava Challenge  - 英文單字拼字遊戲

這是一個專為台灣國中生設計的英文單字練習小遊戲。透過落下的單字泡泡，以遊戲化的方式幫助使用者記憶單字。無需安裝任何軟體，打開瀏覽器即可遊玩！

🎮 **[點此立即遊玩 (Play Game)](https://howlowkiki.github.io/Guava-Challenge/)**

## ✨ 特色 (Features)
* **雙單字庫切換**：
    * 📘 **國中必背 1200 單字**：基礎練習。
    * 📕 **進階挑戰 2000 單字**：更豐富的詞彙量。
* **不重複出題系統**：使用抽牌機制，保證每次遊玩單字不重複出現，直到題庫用盡。
* **三種難度模式**：
    * 🟢 **簡單**：單字緩緩落下，適合初學者。
    * 🟡 **普通**：適中的速度。
    * 🔴 **困難**：挑戰你的打字極限。
* **學習模式**：
    * 答錯或單字落地時，遊戲會自動暫停並顯示正確答案。
    * 玩家可以從容地背誦單字後，按下 `Enter` 鍵繼續挑戰。
* **貼心操作**：
    * 自動鎖定輸入框，打開網頁直接打字。
    * 支援 `Esc` 鍵放棄當前題目。
    * 支援 `Backspace` 鍵 (當輸入框為空時) 快速喚出選單確認視窗。

## 🕹️ 如何遊玩 (How to Play)

1.  **選擇單字庫**：在主選單選擇要挑戰 1200 字或 2000 字。
2.  **選擇難度**：點擊簡單、普通或困難開始遊戲。
3.  **開始打字**：
    * 看著落下的泡泡（顯示中文）。
    * 在輸入框打出對應的英文單字。
    * **不需按 Enter 送出**，拼對會自動得分並消除泡泡。
4.  **特殊控制**：
    * `Esc`：不知道答案時，按 Esc 直接顯示答案（該題失敗）。
    * `Enter`：當答錯暫停時，按 Enter 繼續下一題。
    * `Backspace`：想中途結束？清空輸入框後按 Backspace 呼叫選單。
